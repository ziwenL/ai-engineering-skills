# Android 到 iOS 复刻 Skill

## 适用场景

- Android 已有功能需要在 iOS 端复刻
- 多端行为需要对齐
- 需要从 Android 实现中提炼平台无关逻辑

## 输入

- Android 已实现代码
- iOS 端现有架构和相似模块
- 已确认的产品需求和交互要求

## 工作流程

1. 阅读 Android 已实现代码。
2. 提取平台无关逻辑，包括用户流程、状态机、接口请求、响应处理、错误处理和埋点。
3. 查找 iOS 相似模块。
4. 按 iOS 架构实现。
5. 输出一致性说明和差异点。

## 输出格式

```text
【Android 现有行为】
【平台无关逻辑】
【iOS 涉及模块】
【需要修改的文件】
【行为一致性要求】
【允许保留的平台差异】
【验证方式】
【风险点】
```

## 约束

- 不机械翻译 Android 代码。
- 行为一致优先于结构一致。
- 证据不足时不要猜测 Android 端真实行为。

## 不适用场景

- Android 端还没有稳定实现
- 单端独有能力开发
- 需求本身尚未确认
