# Windows Compatibility

本仓库脚本优先支持 Python，兼容 Windows / macOS / Linux。

Windows 推荐：

```powershell
python scripts\init_submodule_project.py "C:\Users\<your-username>\Projects\your-business-project"
python scripts\check_installation.py "C:\Users\<your-username>\Projects\your-business-project"
```

不要依赖 `rsync`。
